# Crimea is Ukraine RAG 🇺🇦

Мінімалістичний RAG (Retrieval-Augmented Generation) асистент на базі **Chainlit** та **LangChain** для швидкого пошуку та відповідей на запитання щодо окупації Криму на основі матеріалів проєкту ["Крим — це Україна"](https://crimea-is-ukraine.org/).

---

## 🛠 Технологічний стек

* **Python 3.12+**
* **uv** — сучасний менеджер пакетів та залежностей
* **Chainlit** — інтерфейс для чат-ботів
* **LangChain** — фреймворк для розробки додатків на базі LLM
* **Chroma DB** — векторна база даних для пошуку релевантного контексту
* **Fluent Manager** — локалізація та підтримка кількох мов (UA, EN)

---

## 🗄 Ініціалізація векторної бази даних (Chroma DB)

> [!IMPORTANT]
> **Без підготовленої бази даних асистент працювати не буде.**
> Векторну базу даних необхідно згенерувати на основі Markdown-файлів контенту сайту:

1. Переконайтеся, що Ollama запущена та завантажено модель ембеддингів (за замовчуванням `mxbai-embed-large`).
2. Запустіть міграційний скрипт для створення бази знань:
   ```bash
   uv run migrations/create_vector_db_by_md.py
   ```

## 🚀 Швидкий старт

### 1. Локальний запуск за допомогою `uv`

1. Переконайтеся, що встановлено [uv](https://github.com/astral-sh/uv).
2. Налаштуйте змінні оточення:
   ```bash
   cp .env.example .env
   nano .env # Зазначте API-ключі та параметри підключення
   ```
3. Установіть залежності та запустіть застосунок:
   ```bash
   uv sync
   uv run chainlit run src/chainlit_app/main.py
   ```

### 2. Запуск через Docker

1. Налаштуйте змінні оточення:
   ```bash
   cp .env.example .env
   nano .env # Зазначте API-ключі та параметри підключення
   ```
2. Запустіть сервіс:
   ```bash
   docker compose up -d
   ```
   Застосунок буде доступний за адресою `http://localhost:8001`.

## ⚙️ Конфігурація (`.env`)

Основні параметри налаштування:

* `LLM__API_KEY` — API-ключ для доступу до LLM.
* `LLM__BASE_URL` — базовий URL для API (наприклад, для Lapathoniia або локального хостингу).
* `LLM__MODEL` — назва моделі (наприклад, `MamayLM-Gemma-3-12B-IT-v1.0`).
* `EMBED__MODEL` & `EMBED__URL` — модель та хост для ембеддингів (за замовчуванням Ollama / `mxbai-embed-large`).

## 📂 Структура проєкту

* `src/chainlit_app/` — логіка роботи чату та взаємодія з інтерфейсом Chainlit.
* `src/core/` — менеджери для роботи з векторною базою даних (Chroma) та моделями.
* `src/config/` — налаштування застосунку за допомогою Pydantic Settings.
* `database/` — векторна база знань (Chroma DB).
* `locales/` — файли локалізації (Fluent `.ftl` шаблони).
